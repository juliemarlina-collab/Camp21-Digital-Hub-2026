Visual Identity Foundation
Design Direction & Mood
The corrected and definitive design direction for Camp 21 2026 website — grounded in the real Kompleks Kraf KL identity.
⚠️ Critical correction from the asset brief: Previous iterations generated futuristic glass buildings, luxury hotels or convention centres for the venue. This is WRONG. The correct identity is "Heritage Village in the Heart of Kuala Lumpur."
✅ Correct Direction — Use These
✓Traditional Malay-style craft complex, pavilion/compound feel
✓Steep red/brown tiled roofs, ochre/yellow facade, carved wooden details
✓Black-and-gold "Kompleks Kraf Kuala Lumpur" signage + flagpoles
✓Tropical landscaping — gardens, warm daylight or golden hour
✓KL high-rises (KLCC, Petronas Towers) subtly in the background
✓Batik canting hands, songket border, wau bulan, ukiran kayu as foreground accents
✓Mood: prestigious, welcoming, cultural, educational, internationally professional
❌ Negative Direction — Avoid These
✗Futuristic glass buildings or towers
✗Luxury hotel or resort aesthetic
✗Shopping mall, convention centre, generic modern museum
✗Dark futuristic night scene with cold neon lighting
✗Generic Asian motifs or non-Malaysian craft references
✗Neon colours or cold corporate grey-only palette
✗Loud/overcrowded patterns that reduce text readability
✗Generic SaaS-style dashboard without Malaysian craft identity

Professional Review
Gap Analysis & Recommendations
Identified gaps between the current asset brief and a professional website outcome, with specific recommendations for each.
Critical Gaps
Critical
No mobile-responsive CSS system specified
The asset brief specifies ratios and placements but contains no responsive breakpoint guidance. The 8 craft cards in a 4-column grid will break on mobile without a defined grid collapse strategy. The hero 16:9 banner will lose legibility below 480px without a separate mobile crop.
→ Add CSS breakpoints: 4-col → 2-col (≤768px) → 1-col (≤480px). Specify separate hero crop at 4:5 for mobile using CSS art-direction on the element.
Critical
Venue visual correction not yet actioned
The spreadsheet explicitly flags that previous generated visuals produced a futuristic glass building. The corrected prompt is written but no new asset has been generated. The entire landing page identity depends on this single hero image being correct.
→ Asset 01 (hero-main.jpg) must be the very first generation using the corrected prompt in the spreadsheet. Test against the negative prompt checklist before proceeding to any other asset.
Major Gaps
Major
Missing: Explore Malaysia section (Asset 13) not in generation order
The original generation order table in the spreadsheet lists only 14 assets but Asset 13 (Explore Malaysia Through Craft, 16:9) is placed at Stage 2 in the category table but omitted from the generation order table entirely. This tourism visual is crucial for the "Explore Malaysia" section.
→ Insert Asset 13 between items 8 and 9 in the Stage 2 queue. It is a High priority asset that supports the cultural tourism positioning.
Major
No loading state / placeholder system defined
All 16 assets are "To be generated" — meaning the website currently has no images at all. Without placeholder colours or skeleton loading states, the site will look broken during development and before assets are ready.
→ Define CSS placeholder backgrounds using the heritage colour palette for each image slot. Use the assigned craft theme colour (e.g. batik card placeholder = terracotta gradient; songket card = gold-navy gradient) so the layout reads as intentional even before images load.
Major
Terracotta/Ochre CSS tokens missing from Smart DT system
The updated colour palette includes --terracotta (#C0522A) and --ochre (#D4821A) derived from the venue's actual architecture, but these colours do not exist in the current Smart DT CSS token set (which uses only navy, teal, orange, yellow, muted). Without adding them, the heritage warmth of the new direction cannot be expressed in the UI components.
→ Add --terracotta:#C0522A, --ochre:#D4821A, --maroon:#7A1530, --cream:#F5EDD8 to the CSS :root variables. Apply terracotta to the venue section accent, ochre to section eyebrow labels on heritage pages.
Suggestions & Enhancements
Suggestion
Add a "Heritage Craft Identifier" SVG icon set
Currently all craft sections rely on generated photographs. An additional set of 5–6 vector SVG icons (batik canting, wau bulan silhouette, songket loom, ukiran motif, anyaman weave, kite tail) would provide scalable, always-available visual identifiers usable at any size — in navigation, section labels, and tooltips — without depending on generated assets loading.
→ Create or source SVG icons in the cream + gold palette for each craft type. These become the universal visual language of the site independent of photo assets.
Suggestion
Add a hero headline tagline as overlaid text — not embedded in the image
The hero prompt specifies "leave clean space on the left for website headline text." This means the headline and CTA buttons must be implemented as HTML/CSS overlay text — NOT embedded in the image. If the tagline is baked into the generated image, it cannot be changed or translated.
→ Implement hero text as absolutely-positioned HTML over the image. Use: "Where traditional craft meets future innovation" as H1 in Poppins 900, white, with a 40% navy scrim behind it for legibility.
Suggestion
Specify WebP format with JPG fallback for all 16 assets
All assets are currently specified as .jpg. Modern browsers support WebP which offers 25–35% smaller file sizes at equal visual quality — critical for the hero (16:9) and venue showcase (16:9) which will be large files.
→ Generate final assets as JPG, then convert to WebP using Squoosh or ImageOptim. Use the HTML tag to serve WebP to supporting browsers with JPG fallback.
Suggestion
Missing: Open Graph / social preview asset
No social sharing / OG image is defined in the asset brief. When the site is shared on WhatsApp, Telegram, or LinkedIn, it will show either nothing or a random image. Given this is a programme website being shared heavily through organiser Telegram groups, this is a visible gap.
→ Add Asset 17: OG Social Preview Image at 1200×630px (close to 2:1). Repurpose the hero composition with the Camp 21 title, date and "Kompleks Kraf KL" text overlaid. Generate as a separate output.
